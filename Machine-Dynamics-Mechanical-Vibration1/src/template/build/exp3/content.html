Introduction


A system is said to be a Fixed-Fixed beam system if it has a fixed connections at both its ends.

Determining the natural frequency of any system helps us to find how the system will behave when just disturbed and left, and to find what kind of excitation frequency to be avoided in the system.
Vibration analysis of a fixed-fixed beam system is important as it can explain and help us analyse a number of real life systems.
The following few real system can be simplified to a fixed-fixed beam,
thereby helping us make design changes accordingly for the most efficient systems.
